Dear internet. I've noticed lately that every great blog CMS seems to go through the same journey:

1. Genesis - a dev says "a blog is just a light WYIWYG editor, data storage and static site generator. I shall make one with no bloat." They then build it, and it's sick.
2. Take off - after using it for themselves they realise, oh...other people want this. I should charge them. They do this, and it's sick.
3. tOo tHe mOon - then the deve realises that to get word out faster they should probably raise some cash, get those ads running and finally add all those features the users want. I'm talking comments, likes, Zapier integration, AI chatbots.
4. The Zuckerberg Dilemma - They realise that their blog site has quite a lot of traffic now, but blogger don't really want to pay enough to reach the deca-unicorn exit the dev now wants/needs. Time to run some ads, heck - we could even allow blogger to paywall content!
5. Repeat.
